## RDS Project Page
How to use:
1. First, clone this repo.
2. Make sure you have the Live Server extension by Ritwick Dey installed in VS Code.
3. When you make changes/push, the actual website will update.
4. Before doing this, test the changes by right clicking `index.html` and pressing `Open with Live Server`
    * This will allow you to view changes before updating the actual webpage.
5. Pretty much all of the changes should only need to be in `index.html`, by either modifying text or adding additional images/videos. Do not modify the following (unless you know what you're doing):
    * The git branch `backup`
    * Anything ending in `.css`
    * Anyting ending in `.js`

To view the actual page, go to: [https://jarmibe7.github.io/rds-powerhouse/
](https://jarmibe7.github.io/rds-powerhouse/
